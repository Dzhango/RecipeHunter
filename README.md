### Installation

1. Get a free API Key at [Spoonacular](https://spoonacular.com/food-api)
2. Clone the repo
   ```sh
   git clone https://github.com/Dzhango/cse-110-team-6
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `config.js`
